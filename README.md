# Merkle Tree Implementation  

To run the program:

1. Open the project folder in your IDE and run the file: "MerkleTree.py"

2. The program will:
   - Generate sample datasets
   - Build multiple Merkle Trees
   - Compare tree hashes to detect modified data
   - Demonstrate 1-bit and 4-bit hash collision behavior

3. Output will be displayed directly in the console.

## Concepts Demonstrated
- SHA-1 hashing
- Merkle Tree construction
- Data integrity verification
- Collision detection
- Recursive tree traversal

This project was developed for cryptography coursework using Python.
